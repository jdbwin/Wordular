# Wordular
*Find and remember your favourite words...*

## Frontend
- React
- Redux
- PropTypes
- css-modules
- Bulma
- Webpack
- Babel

## Backend
- Node
- Koa

## Setup
1. Clone repository:  
`git clone https://github.com/jdbwin/wordular.git`

2. Install packages:  
`yarn`  

3. Run:  
`yarn start`

4. Wordnik:
  - Get a free Wordnik API key from:  
  https://developer.wordnik.com/  
  
  - Add Wordnik API key to `.env`:  
  `WORDNIK_KEY=XXX`
